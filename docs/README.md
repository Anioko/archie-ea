# Archie Docs — Enterprise Architecture Guides

Practical, vendor-neutral guides for enterprise architects, written by the team behind
**[Archie](https://github.com/Anioko/archie-ea)** — the open-source enterprise
architecture platform (TOGAF 9.2 · ArchiMate 3.2).

## Guides

- **[How to run an Architecture Review Board (ARB)](architecture-review-board.md)** —
  the purpose, cadence, roles, decision records, and a readiness checklist.
- **[ArchiMate 3.2 cheat sheet](archimate-3-2-cheat-sheet.md)** — every layer, the core
  elements, and the relationships, in one reference.
- **[Application portfolio rationalization (TIME / 6R)](application-rationalization.md)** —
  how to assess and rationalize an application landscape.
- **[TOGAF ADM with ArchiMate](togaf-adm-with-archimate.md)** — how the two standards fit
  together, phase by phase.
- **[Open-source enterprise architecture tools](open-source-enterprise-architecture-tools.md)** —
  the landscape, and where an open-source platform fits.

> These guides are framework guidance you can apply with any tool. Where a guide shows
> *how Archie does it*, that part is optional — the practice stands on its own.
