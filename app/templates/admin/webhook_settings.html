Setup Instructions

Microsoft Teams

  1. Open the channel you want to post notifications to.
  2. Click ··· → Connectors → search Incoming Webhook.
  3. Click Configure, give it a name, then click Create.
  4. Copy the generated URL and paste it into the Webhook URL field above.

Slack

  1. Go to api.slack.com/apps and create or select your app.
  2. Navigate to Features → Incoming Webhooks and enable it.
  3. Click Add New Webhook to Workspace and select a channel.
  4. Copy the Webhook URL and paste it above.

Payload Formats

Teams subscriptions send Adaptive Card v1.4 messages with a bold event title, structured field list, and timestamp.

Slack subscriptions send Block Kit messages with a header block, mrkdwn section, and context block timestamp.

Generic subscriptions send the raw event JSON payload as-is.
