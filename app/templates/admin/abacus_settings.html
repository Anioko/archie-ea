Setup Instructions

  1. Log in to your Abacus instance as an administrator
  2. Navigate to Settings → API → OAuth Applications
  3. Create a new OAuth application with the following settings:
    • Grant type: client_credentials
    • Scopes: read:objects read:relationships read:architectures
  4. Copy the Client ID and Client Secret
  5. Paste them into the fields above and configure sync settings
  6. Click Test Connection to verify the setup
  7. Enable integration and auto-sync when ready

For more information, see the Abacus Integration Guide.
